\doxysubsection{Descrição detalhada}
Função principal do sistema de gerenciamento de jogos. 

Este arquivo contém a função {\ttfamily main}, que é o ponto de entrada do programa. A função gerencia a interação com o usuário, exibindo um menu de opções e executando as funcionalidades correspondentes, como cadastrar jogadores, remover jogadores, listar jogadores e executar partidas.\hypertarget{main_8cpp_autotoc_md0}{}\doxysubsubsection{\texorpdfstring{Fluxo do Programa\+:}{Fluxo do Programa\+:}}\label{main_8cpp_autotoc_md0}

\begin{DoxyEnumerate}
\item Exibe um menu de opções para o usuário.
\item Lê a entrada do usuário e valida o comando.
\item Executa a funcionalidade correspondente ao comando\+:
\begin{DoxyItemize}
\item {\bfseries{CJ}}\+: Cadastra um novo jogador.
\item {\bfseries{RJ}}\+: Remove um jogador existente.
\item {\bfseries{LJ}}\+: Lista todos os jogadores cadastrados.
\item {\bfseries{EP}}\+: Executa uma partida em um dos jogos disponíveis.
\item {\bfseries{FS}}\+: Finaliza o sistema.
\end{DoxyItemize}
\item Repete o processo até que o usuário escolha a opção de finalizar o sistema.
\end{DoxyEnumerate}\hypertarget{main_8cpp_autotoc_md1}{}\doxysubsubsection{\texorpdfstring{Dependências\+:}{Dependências\+:}}\label{main_8cpp_autotoc_md1}

\begin{DoxyItemize}
\item Utiliza a classe {\ttfamily \doxylink{class_central_de_jogos}{Central\+De\+Jogos}} para gerenciar jogadores e partidas.
\item Utiliza funções auxiliares como {\ttfamily validar\+Entrada} e {\ttfamily exibir\+Menu} para interação com o usuário.
\end{DoxyItemize}
