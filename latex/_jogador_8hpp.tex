Definição da classe \doxylink{class_jogador}{Jogador}. 

Esta classe representa um jogador no sistema, armazenando informações como nome, apelido e estatísticas em diferentes jogos. As estatísticas são gerenciadas por meio de um mapa que associa o nome do jogo a um objeto da classe {\ttfamily \doxylink{class_estatisticas}{Estatisticas}}, permitindo o registro e consulta de vitórias, derrotas e empates para cada jogo individualmente.\hypertarget{_jogador_8hpp_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{Atributos\+:}{Atributos\+:}}\label{_jogador_8hpp_autotoc_md7}

\begin{DoxyItemize}
\item {\ttfamily apelido}\+: Apelido único do jogador, usado para identificação.
\item {\ttfamily nome}\+: Nome completo do jogador.
\item {\ttfamily estatisticas\+Por\+Jogo}\+: Mapa que armazena as estatísticas do jogador para cada jogo. A chave é o nome do jogo (por exemplo, "{}\+VELHA"{}, "{}\+LIG4"{}, "{}\+REVERSI"{}, "{}\+BATALHANAVAL"{}), e o valor é um objeto da classe {\ttfamily \doxylink{class_estatisticas}{Estatisticas}}.
\end{DoxyItemize}\hypertarget{_jogador_8hpp_autotoc_md8}{}\doxysubsubsection{\texorpdfstring{Funcionalidades Principais\+:}{Funcionalidades Principais\+:}}\label{_jogador_8hpp_autotoc_md8}

\begin{DoxyItemize}
\item Registro de vitórias, derrotas e empates para um jogo específico.
\item Consulta das estatísticas (vitórias, derrotas e empates) para um jogo específico.
\item Exibição das estatísticas de um jogo específico.
\end{DoxyItemize}\hypertarget{_jogador_8hpp_autotoc_md9}{}\doxysubsubsection{\texorpdfstring{Integração com Outras Classes\+:}{Integração com Outras Classes\+:}}\label{_jogador_8hpp_autotoc_md9}
