Definição da classe \doxylink{class_jogos}{Jogos}. 

Esta classe é uma classe base abstrata que define a interface comum para todos os jogos do sistema. Ela fornece métodos e atributos básicos para gerenciar partidas, como manipulação do tabuleiro, verificação de vitória e empate, e controle de turnos. Classes derivadas, como {\ttfamily \doxylink{class_jogo_da_velha}{Jogo\+Da\+Velha}}, {\ttfamily \doxylink{class_lig4}{Lig4}}, {\ttfamily \doxylink{class_reversi}{Reversi}} e {\ttfamily \doxylink{class_batalha_naval}{Batalha\+Naval}}, implementam as funcionalidades específicas de cada jogo.\hypertarget{_jogos_8hpp_autotoc_md12}{}\doxysubsubsection{\texorpdfstring{Funcionalidades Principais\+:}{Funcionalidades Principais\+:}}\label{_jogos_8hpp_autotoc_md12}

\begin{DoxyItemize}
\item Gerenciamento de tabuleiros genéricos.
\item Controle de turnos entre jogadores.
\item Verificação de vitória e empate (a serem implementados pelas classes derivadas).
\item Exibição do tabuleiro e anúncio de turnos.
\item Sorteio de quem começa a partida.
\end{DoxyItemize}\hypertarget{_jogos_8hpp_autotoc_md13}{}\doxysubsubsection{\texorpdfstring{Métodos Virtuais Puros\+:}{Métodos Virtuais Puros\+:}}\label{_jogos_8hpp_autotoc_md13}

\item {\ttfamily anunciar\+Inicio\+Partida}\+: Anuncia o início da partida e define o jogador que começa (deve ser implementado pelas classes derivadas).
