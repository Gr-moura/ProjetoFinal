Definição da classe \doxylink{class_central_de_jogos}{Central\+De\+Jogos}. 

A classe \doxylink{class_central_de_jogos}{Central\+De\+Jogos} é o núcleo do sistema de gerenciamento de jogos e jogadores. Ela é responsável por\+:
\begin{DoxyItemize}
\item Gerenciar uma lista de jogadores cadastrados, incluindo suas estatísticas em diferentes jogos.
\item Controlar as instâncias dos jogos disponíveis\+: Jogo da Velha, \doxylink{class_lig4}{Lig4}, \doxylink{class_reversi}{Reversi} e Batalha Naval.
\item Facilitar a execução de partidas, permitindo que os jogadores escolham um jogo e compitam entre si ou contra uma inteligência artificial (no caso do Jogo da Velha).
\item Fornecer funcionalidades para cadastrar, remover, ordenar e listar jogadores.
\end{DoxyItemize}

A classe atua como uma centralizadora de operações, conectando os jogadores aos jogos e garantindo que as estatísticas sejam atualizadas corretamente após cada partida.\hypertarget{_central_de_jogos_8hpp_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{Funcionalidades Principais\+:}{Funcionalidades Principais\+:}}\label{_central_de_jogos_8hpp_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Gerenciamento de Jogadores}}\+:
\begin{DoxyItemize}
\item Cadastro de novos jogadores com nome e apelido.
\item Remoção de jogadores existentes.
\item Listagem de todos os jogadores cadastrados.
\item Ordenação dos jogadores por critérios específicos (por exemplo, nome ou apelido).
\end{DoxyItemize}
\item {\bfseries{Gerenciamento de \doxylink{class_jogos}{Jogos}}}\+:
\begin{DoxyItemize}
\item Execução de partidas em qualquer um dos jogos disponíveis.
\item Atualização automática das estatísticas dos jogadores após cada partida.
\item Interação com a inteligência artificial (AI) no Jogo da Velha para partidas single-\/player.
\end{DoxyItemize}
\item {\bfseries{Persistência de Dados}}\+:
\begin{DoxyItemize}
\item Carregamento dos dados dos jogadores a partir de um arquivo de texto ao inicializar o sistema.
\item Salvamento dos dados dos jogadores em um arquivo de texto ao finalizar o sistema, garantindo que as estatísticas sejam mantidas entre execuções.
\end{DoxyItemize}
