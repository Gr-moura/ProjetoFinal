\hypertarget{md__u_i_2_r_e_a_d_m_e_autotoc_md1}{}\doxysection{\texorpdfstring{Objetivo}{Objetivo}}\label{md__u_i_2_r_e_a_d_m_e_autotoc_md1}
O objetivo da UI do projeto é melhorar a experiância do usuárioa ao interagir com o código em c++. ~\newline
 O código em c++ pode ser tanto rodado usando a UI como diretamente no terminal. Para executar o código com a UI, siga as instruções disponóveis mais a frente. ~\newline
\hypertarget{md__u_i_2_r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Solução}{Solução}}\label{md__u_i_2_r_e_a_d_m_e_autotoc_md2}
A solução usada para desenvolver a UI foi o REACT. Usando código em javascript, desenvolvemos uma interface gráfica para o projeto que pode ser executada diretamente em qualquer navegador. ~\newline
 Podemos dividir, de uma forma geral, a UI entre telas e componentes. ~\newline
 \hypertarget{md__u_i_2_r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{\texorpdfstring{O worflow REACT}{O worflow REACT}}\label{md__u_i_2_r_e_a_d_m_e_autotoc_md3}
O react trabalha usando html, css, e javascript para criar páginas web. ~\newline
 O diferencial que o usa do REACT proporciona é a capacidade de criar "{}componentes"{}\+: Funções que podem ser chamadas como tags html. A criação de componentes proporciona uma facilidade imensa em criar partes reutilizáveis para o site. ~\newline
 Além disso, o REACT proporciona diversos outros beneficios, como a renderização condicional (usando "{}state variables"{} é possível re-\/renderizar apenas as partes do site que sofreram mudanças, melhorando o desempenho da aplicação) e as routes (forma de navegar entre as páginas do site sem ter que recarregar a janela). \hypertarget{md__u_i_2_r_e_a_d_m_e_autotoc_md4}{}\doxysubsection{\texorpdfstring{Telas}{Telas}}\label{md__u_i_2_r_e_a_d_m_e_autotoc_md4}

\hypertarget{md__u_i_2_r_e_a_d_m_e_autotoc_md6}{}\doxysection{\texorpdfstring{Como rodar?}{Como rodar?}}\label{md__u_i_2_r_e_a_d_m_e_autotoc_md6}
1-\/ Instale o package managaer NPM na sua máquina ~\newline
 2-\/ Abra um terminal no diretório UI ~\newline
 2-\/ Digite {\ttfamily npm install} e pressione enter ~\newline
 3-\/ No mesmo terminal, digite {\ttfamily npm run dev} ~\newline
 4-\/ Copie o link resultante e cole-\/o em um navegador de sua escolha ~\newline


IMPORTANTE\+: não feche a janela do terminal enquanto estiver usando a UI ~\newline
